```bash
# No install required — run directly with npx
npx josfox serve

npx josfox run artifact.jos

```

## What This Is

This package is a tiny wrapper (~400 bytes) that forwards all commands to [@josfox/jos](https://www.npmjs.com/package/@josfox/jos).

It exists so you can use the memorable `npx josfox` instead of `npx @josfox/jos`.
